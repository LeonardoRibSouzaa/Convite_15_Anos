# 💜 Convite 15 Anos - Eloah

Um site elegante e interativo para convite de aniversário de 15 anos com animações sofisticadas, efeitos visuais incríveis e experiência imersiva.

## 🎨 Características

✨ **Animações Avançadas:**
- Título SVG com animação de escrita em tempo real
- Partículas flutuantes mágicas
- Estrelas piscantes
- Efeito de fumaça animada
- Botões com efeitos hover sofisticados
- Card com glow animado

🎵 **Áudio Inteligente:**
- Música de fundo ambiente
- Pausa automática ao sair da aba
- Controle sensível ao foco da janela

📱 **Responsivo:**
- Funciona perfeitamente em desktop, tablet e mobile
- Background positioning otimizado para mostrar a imagem

🎭 **Narrativa Interativa:**
- Menu navegável
- História imersiva em múltiplas etapas
- Confirmação de presença
- Lista de sugestões de presentes

## 🚀 Como Usar

### Localmente com Python 3:
```bash
cd "15-Anos"
python -m http.server 8000
```
Acesse: `http://localhost:8000`

### Com Node.js:
```bash
npx http-server
```

### Com VS Code Live Server:
1. Instale a extensão "Live Server"
2. Clique em "Go Live"

## ⚙️ Otimizações Implementadas

- ✅ GPU acceleration (will-change, transform: translateZ)
- ✅ Backface visibility para melhor performance
- ✅ Background position otimizado
- ✅ Lazy loading de animações
- ✅ Event listeners eficientes
- ✅ Pausas automáticas de áudio

## 🎵 Controle de Áudio

- **Toca ao:** Clicar em qualquer lugar da página
- **Pausa ao:** Sair da aba/janela
- **Para ao:** Fechar ou deixar o site
- **Volume:** 20% para conforto

## 📱 Responsividade

- **Desktop:** Experiência completa com todas as animações
- **Tablet:** Layout ajustado mantendo qualidade
- **Mobile:** Interface adaptada com botões otimizados

### Breakpoints:
- `≤480px` - Mobile
- `≤768px` - Tablet
- `>768px` - Desktop
